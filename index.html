Parameter Value Description

Biomarker Selection Criteria

Upregulated Genes

  • Filtered on mean expression
  • avglogFC > 1.5
  • Malignant.pct > 50%
  • Top 50 genes by p-val-adj

Downregulated Genes

  • Filtered on mean expression
  • avglogFC < -1
  • Control.pct > 50%
  • Top 50 genes by p-val-adj

Methods & Computational Tools

Comprehensive overview of computational approaches and software tools used in the analysis

LIBRA

Library for Integrated Biological Research Analysis

Key Features:

  • Pseudobulk differential expression analysis
  • DESeq2 integration with LRT testing
  • Single-cell RNA-seq data processing
  • Batch effect correction

Used for:

Biomarker Analysis Signature Analysis

DESeq2

Differential gene expression analysis based on negative binomial distribution

Key Features:

  • Likelihood Ratio Test (LRT)
  • Multiple testing correction
  • Robust statistical modeling
  • Pseudobulk aggregation support

Used for:

Statistical Testing P-value Calculation

ClusterProfiler

Statistical analysis and visualization of functional profiles for genes and gene clusters

Key Features:

  • Over-representation analysis
  • Gene set enrichment analysis
  • Multiple database integration
  • Autocompare functionality

Used for:

Pathway Analysis Functional Enrichment

Fgsea

Fast Gene Set Enrichment Analysis

Key Features:

  • Fast preranked GSEA
  • Adaptive multilevel splitting
  • Leading edge analysis
  • Multiple testing correction

Used for:

Ranked Gene Lists Pathway Enrichment

Seurat

Single-cell RNA sequencing analysis toolkit

Key Features:

  • FindMarkers function
  • Wilcoxon rank sum test
  • Cluster annotation
  • Cell type identification

Used for:

Cell Line Analysis Cluster Markers
